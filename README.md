# 臓器パズルゲーム 🎮

60秒のタイムアタック型パズルゲーム！同じ臓器を繋げて消していこう。

## デモ

Vercelでホスティング中：[あなたのVercel URL]

## 🎯 ゲームの特徴

- **タイムアタック**: 60秒間の制限時間
- **コンボシステム**: 連続で消すと高得点！
- **6種類の臓器**: 心臓、脳、肺、腎臓、肝臓、骨
- **パーティクルエフェクト**: 爽快な演出
- **揺れアニメーション**: ブロックがゆらゆら動く

## 🎮 遊び方

1. **開始**: 「ゲームスタート」ボタンをクリック
2. **操作**: 同じ色（臓器）のブロックをドラッグで繋げる
3. **目標**: 60秒以内にできるだけ高いスコアを目指す

## 🚀 技術スタック

- React 18
- Vite
- Canvas API
- Vercel（ホスティング）
